REST API

Open postman or any other api testing app to test the routes.

API Endpoints

List of available routes:

Auth routes:
POST api/auth/signup - Signup
POST api/auth/login - Signin

Passowds routes:
POST api/passwords/ - Create a Passowds with website name
GET api/passwords - Get all website names with there password in encrypted
GET api/passwords/:id - Get a particular website name with password in decrypted manner
PUT api/passwords/:id - Update website name and password
DELETE api/passwords/:id - Delete website name with password
